# Project Objective | Objetivo do projeto
This project is a CRUD application using Flask for a restaurant menu. The purpose of the project is to improve API development practices with the Python language and continuously enhance it.

Este projeto é um CRUD usando Flask de um cardápio para restaurantes. A finalidade é do projeto está em melhorar práticas de desenvolvimento de API com a linguagem Python e sempre aperfeiçoá-lo.

# How to Run Locally | Como rodar localmente
- Make sure you have Poetry installed on your machine
- Have a compatible version of Python 3.13 or higher installed
- To install the project dependencies: `poetry install`
- To start the API: `poetry run flask --app api_cardapio run --debug`
- The API URL will be: http://127.0.0.1:5000/swagger/
- Test it
